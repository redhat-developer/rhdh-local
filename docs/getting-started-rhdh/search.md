# Using Search

Search is one of the most powerful features in Red Hat Developer Hub. It allows you to quickly find components, services, APIs, documentation, templates, and more across your entire software ecosystem.

Developer Hub provides a quick search from the Global Header that shows instant results. When you need more detailed results with advanced filtering options, you can click the **"All results..."** link to view the dedicated search results page.

!!! success "Universal Search Across Everything"
    Developer Hub search indexes and searches across:

    * Software components and services
    * APIs and API definitions
    * Technical documentation (TechDocs)
    * Software templates
    * Teams, users, and groups
    * Any other resources in your catalog

## From the header

![Red Hat Developer Hub Global Header](../images/globalheader.png)

The **Search input field** in the Global Header provides instant, universal search across all resources in Developer Hub. It's always accessible and perfect for quick lookups.

### Quick Search Features

**Instant Results**

* Start typing and see results appear immediately
* Results update as you type, helping you refine your search
* No need to press Enter or click a search button

**Smart Suggestions**

* Developer Hub suggests relevant results as you type
* See previews of components, documentation, and APIs
* Click any suggestion to navigate directly to that resource

### Using Header Search

1. **Click or focus** on the search field in the Global Header
2. **Start typing** your search term
3. **Review results** as they appear in the dropdown
4. **Click a result** to navigate directly to that resource, or click **"All results..."** to view the full search results page with advanced options
5. **Clear your search** using the Clear button (X icon) in the search field to reset and start a new search

!!! tip "Search Tips for Header Search"
    * Use specific terms: "payment-service" is better than "payment"
    * Search by owner: Try searching for team names or usernames
    * Search by type: Include terms like "API", "service", or "template"
    * Use partial matches: Developer Hub finds partial matches automatically

### Search Result Types

When you search from the header, you'll see different types of results:

**Components**

* Services, websites, libraries, and other software components
* Shows component name, description, owner, and lifecycle status
* Click to view the component's detail page

**APIs**

* API definitions and specifications
* Shows API name, description, and owner
* Click to view API documentation and test endpoints

**Documentation**

* TechDocs and other documentation pages
* Shows document title and a brief excerpt
* Click to open the documentation

**Templates**

* Software templates for creating new projects
* Shows template name and description
* Click to start using the template

**Users and Groups**

* People and teams in your organization
* Shows name, email, and role information
* Click to view profile or team details

## Viewing All Search Results

After performing a quick search from the Global Header, you can access the dedicated **Search Results** page by clicking the **"All results..."** link. This full-page interface provides advanced filtering, result categorization, and detailed views of all matching resources.

### Accessing the Search Results Page

1. **Perform a search** using the search field in the Global Header
2. **Click the "All results..." link** that appears in the search dropdown
3. You'll see a full-page search interface with categorized results and advanced filtering options

### Advanced Search Features

The search results page provides several powerful features to help you find exactly what you need:

**Filtered Results**

* Results are organized by Result Type (Catalog items or Documentation)
* Click on a category to see only those results
* Reset your selection by clicking "All" result types

**Search Filters**

* Filter by resource Types (Software Catalog, Documentation) and Kinds (Component, API, Documentation, Template, etc.)
* Filter by lifecycle status (Experimental, Production, etc.)
* Useful when you know what kind of resource you're looking for

**Detailed Result Views**

* Click on any result to see more information, relationships, and dependencies
* Access quick actions directly from search results
* Use the star icon to save items to your Starred items for quick access

**Pagination** (Available only on the "All results" page)

* Control how many results are displayed per page: 10, 25, 50, or 100 results
* Use the arrow buttons to navigate between pages of results
* Adjust the results per page to match your preference and screen size

**Clear Search**

* Use the **Clear** button in the search term box to reset your search
* Quickly start a new search without manually deleting the search term

### Using the Search Results Page

Once you've clicked "All results..." from the header search, you'll see the full search results page. Here's a typical workflow:

1. **Review results** - Results are automatically ranked by relevance
2. **Apply filters** - Use the filter options to narrow down results by type, kind, or lifecycle status
3. **Navigate results** - Use pagination controls to browse through multiple pages of results
4. **Click a result** - View full details, relationships, and access quick actions
5. **Refine your search** - Edit the search term at the top or use the Clear button to start fresh

!!! success "Powerful Search Capabilities"
    The dedicated search page is ideal when you need to:
    
    * Find multiple related resources
    * Explore what's available in a specific category
    * Use advanced filtering to narrow results

### Search Best Practices

**Be Specific**

* Use exact component names when you know them
* Include relevant keywords from descriptions
* Combine terms for better precision

**Use Filters Strategically**

* Start broad, then narrow with filters
* Use type filters when searching for a specific resource category
* Combine multiple filters for precise results

**Explore Related Resources**

* Click on an item's owners to see their other resources
* Use tags to discover related components
* Follow relationships between resources

**Save Time with Bookmarks and Stars**

* Bookmark important search results in your browser for quick access
* Star frequently used components to add them to your Starred items
* Use browser bookmarks to save common search queries

### Search Shortcuts and Tips

**Search Operators**

* Developer Hub supports natural language search
* Use quotes for exact phrases: `"RHDH Local"`
* Combine terms: `api payment`

**Understanding Results**

* Results are ranked by relevance
* More specific matches appear first
* Recent updates may boost relevance
* Owner and team information helps identify the right resource

### Troubleshooting Search

**Not Finding What You're Looking For?**

1. **Check spelling**: Typos can prevent matches
2. **Try broader terms**: Be less specific if results are empty
3. **Check filters**: Make sure filters aren't excluding your results
4. **Try different keywords**: Use synonyms or related terms
5. **Search by owner**: If you know who owns it, search for the owner first

**Results Seem Incomplete?**

* Some resources may not be indexed yet
* Check if the resource exists in the Catalog
* Contact your administrator if resources are missing
* Some resources may be restricted by permissions
