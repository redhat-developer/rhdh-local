When you first open Red Hat Developer Hub, you'll notice two main navigation areas: the **Global Header** at the top and the **Left Hand Navigation** sidebar. Understanding these areas is key to efficiently using Developer Hub.

!!! note "RHDH vs. Backstage UI"
    While Red Hat Developer Hub shares many UI similarities with CNCF Backstage, the Global Header in RHDH has been customized with additional features and components specific to the Red Hat Developer Hub experience.

## Using the Global Header

The Global Header is your primary toolbar, always visible at the top of the screen. It provides quick access to the most commonly used features and functions in Developer Hub.

![Red Hat Developer Hub Global Header](../images/globalheader.png)

### Self-Service Button (+)

The **Self-Service button** (displayed as a `+` icon) is your gateway to creating new resources quickly and efficiently.

!!! success "Quick Access to Templates"
    Click the `+` button to access a variety of templates that enable you to:

    * Bootstrap new projects from within Developer Hub
    * Set up new services (if your organization allows this)  

This button streamlines the process of starting new work by providing instant access to your organization's approved templates and workflows. No need to navigate through multiple menusâ€”everything you need to create is right at your fingertips.

### Support Button

The **Support button** provides direct access to help resources when you need assistance.

!!! tip "Customizable Support Link"
    Your organization can configure this button to link to either:

    * An internal support page or ticketing system
    * An external support resource
    * Documentation or help center

Whether you need to report an issue, ask a question, or find troubleshooting information, the Support button ensures help is always just one click away.

### Notifications Button

Stay informed with the **Notifications button**, which displays alerts and updates from:

* Plugins and extensions
* External services integrated with Developer Hub
* System updates and announcements
* Workflow status changes

Click the notifications icon to view your alerts and stay up-to-date with what's happening across your development ecosystem.

### Search Input Field

The **Search input field** in the Global Header is one of the most powerful features in Developer Hub.

!!! success "Universal Search"
    Use the search field to quickly find:

    * Services and components
    * Documentation and TechDocs
    * APIs and API definitions
    * Templates
    * Teams and users
    * Any other resources within Developer Hub

Simply start typing in the search field, and Developer Hub will provide instant results across all available resources. This makes finding what you need faster than navigating through multiple pages.

For more details on using search effectively, see the [Using Search](search.md) guide.

### Quick Links To Other Applications

The Global Header includes **quick links to other applications** that provide access to a variety of other applications in your organization.

!!! info "Quick Links To Other Applications"
    This feature helps you:

    * Quickly navigate away to other tools and dashboards 

Quick links support your productivity, connecting you with your CI/CD tools, monitoring systems, project management platforms, and more. The Global Header makes accessing these systems quick and easy.

### User Profile Drop-Down Menu

Your **User Profile** menu (typically accessed via your avatar or name) provides access to:

* **Profile Settings**: Update your personal information and preferences
* **Appearance Customization**: Adjust themes, display settings, and UI preferences
* **Developer Hub Metadata**: View information about your Developer Hub instance, version, and configuration
* **Logout**: Securely sign out of Developer Hub

This menu puts you in control of your Developer Hub experience, allowing you to personalize the interface to match your preferences and workflow.

## Using the Left Hand Navigation

The **Left Hand Navigation** sidebar provides access to the main sections and features of Developer Hub. While the Global Header focuses on quick actions and tools, the sidebar is your primary navigation structure.

### Standard Navigation Items

The left sidebar typically includes these core sections:

**Home**

* Your Developer Hub homepage with overviews, recent activity, and quick links

**Catalog**

* Browse and manage all your software components, services, APIs, and resources
* Filter, search, and organize your software ecosystem
* View component details, relationships, and ownership

**Docs** (TechDocs)

* Access all technical documentation
* View documentation for components and services
* Navigate through your organization's knowledge base

**APIs**

* Access software APIs for creating new integrations
* Test APIs right from your browser.

**Administration (Admins Only)**

* Configure your RHDH instance
* Manage permissions and roles
* Browse and install Extensions (plugins)

### Plugin-Added Navigation Items

As plugins are enabled in your Developer Hub instance, they may add their own navigation items to the left sidebar. These could include:

* AI tools like Developer Lightspeed
* Bulk Import to fill your software catalog quickly
* Learning Paths for training and learning
* Custom tools and workflows specific to your organization

### Navigation Best Practices

Make the most of Developer Hub by following these tips to streamline your workflow and improve your productivity:

!!! tip "Bookmark Frequently Used Items"
    Save time by keeping your most-accessed resources at your fingertips:

    * Use your browser's bookmarking feature for components or pages you access regularly
    * Star frequently used components to add them to your Starred items for quick access
    * Use the search function to quickly return to recently viewed resources

!!! tip "Customize Your View"
    Personalize Developer Hub to match your preferences and workflow:

    * Use the User Profile menu to adjust appearance settings, themes, and display preferences
    * Customize what information is displayed in different views
    * Adjust settings to highlight the information most relevant to your work

!!! tip "Stay Organized"
    Keep your software ecosystem organized and easy to navigate:

    * Use the Catalog filters to organize components by team, type, or tag
    * Create personal collections or favorites for quick access to important resources
    * Leverage search filters to quickly find what you need across your organization

### Getting Help

If you're ever unsure about where to find something:

1. **Use the Search**: The search function in the Global Header is often the fastest way to find what you need
2. **Use Developer Lightspeed**: Let AI help you find answers to common problems (if installed)
3. **Check the Catalog**: Most resources are accessible through the Catalog section
4. **Review Documentation**: The Docs section contains guides and documentation
5. **Contact Support**: Use the Support button in the Global Header for assistance
